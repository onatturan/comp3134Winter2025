1. What is the definition of a server port?
   A server port is a communication endpoint for networking processes.

2. What is the server port range?
   The range is from 0 to 65535.

3. How many bits does the number above represent?
   It represents 16 bits.

4. State 10 common server ports:
   - 20 (FTP Data)
   - 21 (FTP Control)
   - 22 (SSH)
   - 23 (Telnet)
   - 25 (SMTP)
   - 53 (DNS)
   - 80 (HTTP)
   - 110 (POP3)
   - 143 (IMAP)
   - 443 (HTTPS)
